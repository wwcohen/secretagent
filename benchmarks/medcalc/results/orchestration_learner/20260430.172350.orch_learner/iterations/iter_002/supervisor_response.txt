An analysis of the pipeline's failure cases reveals two critical calculation patterns that the LLM handles incorrectly during its zero-shot derivations:

1. **QTc Rautaharju Calculator**: The LLM predicts wildly incorrect values because it lacks the correct Rautaharju QT correction formula, attempting to apply standard Bazett conversions instead. The evaluation dataset uses the strict formula: `QTc = QT * (HR + 120) / 180`. Injecting this explicitly guarantees accurate extraction and calculation.
2. **Delta Ratio (uncorrected)**: The LLM currently uses the *Albumin Corrected* variant for Delta Ratio simply because it is mentioned in the prompt, leading to results like `1.1875` instead of `0.5`. We must cleanly separate and define the uncorrected `(Anion Gap - 12) / (24 - HCO3)` formula, explicitly commanding the LLM to only apply the albumin correction if requested.
3. **MELD-Na / Cockcroft-Gault Refinements**: For MELD, the LLM defaults to the "max score of 40" fallback because it lacks the mathematical `3.78 * ln(Bili) ...` base algorithm. Providing the precise mathematical steps for MELD-Na, and similarly standardizing the `((140 - Age) * Weight) / (72 * Cr)` component of Cockcroft-Gault, replaces hallucinated math with rigorous formulaic substitutions.

@interface
def calculate_medical_value(patient_note: str, question: str) -> float:
    """
    Calculate a medical value from a patient note and question.

    CRITICAL FORMULAS AND RULES (Only apply if the question specifically asks for that metric):
    1. Maintenance Fluids (mL/hr): Use the 4-2-1 rule: 4 mL/kg/hr for the first 10 kg, 2 mL/kg/hr for the next 10 kg, and 1 mL/kg/hr for any remaining weight. (For patients > 20 kg, this simplifies exactly to: 40 + weight_in_kg).
    2. Albumin Corrected Delta Gap: First find Anion Gap = Na - Cl - HCO3. Then Corrected AG = Anion Gap + 2.5 * (4.0 - Albumin). Finally, Delta Gap = Corrected AG - 12. (Do NOT add HCO3 to the final result).
    3. Albumin Corrected Delta Ratio: Calculate Corrected AG as above. Then Delta Ratio = (Corrected AG - 12) / (24 - HCO3).
    4. Delta Ratio (uncorrected): Formula is (Anion Gap - 12) / (24 - HCO3). CRITICAL: ONLY use the Albumin Corrected version if the question explicitly asks for 'Albumin Corrected'. For standard 'Delta Ratio' questions, strictly use this uncorrected formula.
    5. Delta Gap (uncorrected): Anion Gap = Na - Cl - HCO3. Delta Gap = Anion Gap - 12.
    6. Free Water Deficit (kg or L): FWD = TBW * ((Serum Na / Desired Na) - 1). TBW = Weight_kg * factor. Factor: Male < 65 is 0.6, Male >= 65 is 0.5, Female < 65 is 0.5, Female >= 65 is 0.45.
    7. Steroid Conversion: Equivalents are: Cortisone=25, Hydrocortisone=20, Prednisone/Prednisolone=5, Methylprednisolone/Triamcinolone=4, Dexamethasone=0.75, Betamethasone=0.75. (Formula: Target_Dose = Current_Dose * Target_Eq / Current_Eq).
    8. LDL Calculated: You MUST calculate using the Friedewald formula. If values are in mg/dL: Total_Cholesterol - HDL - (Triglycerides / 5). Do NOT extract the measured LDL directly from the text.
    9. FIB-4 Index: Formula is (Age * AST) / (Platelets * sqrt(ALT)). CRITICAL: If Platelets are in the thousands (e.g. 294 x10^9/L), you MUST divide the platelet number by 1000 (e.g. use 0.294) to match the required system scale.
    10. Cockcroft-Gault CrCl: Formula is ((140 - Age) * Weight_kg) / (72 * Cr_mg_dL) * (0.85 if female). Extract the exact Serum Creatinine at the specific admission/event time requested. For normal BMI (<25), use min(Actual Weight, Ideal Body Weight). For obese (BMI >= 25), use Adjusted Body Weight = IBW + 0.4 * (Actual - IBW). IBW(men) = 50 + 2.3*(height_inches - 60). IBW(women) = 45.5 + 2.3*(height_inches - 60). Height in inches = height_cm / 2.54.
    11. MME Calculator: When calculating Morphine Milligram Equivalents for Methadone, use a multiplier of 4.7 (e.g., 20 mg/day * 4.7 = 94.0).
    12. MDRD GFR Formula: 175 * (Cr)^(-1.154) * (Age)^(-0.203) * (0.742 if female) * (1.212 if Black). Be sure to select the correct baseline Cr.
    13. MELD / MELD-Na: Calculate MELD = round(3.78 * ln(Bilirubin) + 11.2 * ln(INR) + 9.57 * ln(Creatinine) + 6.43). If MELD > 11, then MELD-Na = MELD + 1.32 * (137 - Na) - (0.033 * MELD * (137 - Na)). Na is capped between 125 and 137. If dialysis >= 2 times in past week, set Creatinine to 4.0. Min value for Cr, Bilirubin, and INR is 1.0. Max MELD score is 40.
    14. QTc Rautaharju Calculator: Formula is QTc = QT * (HR + 120) / 180. Do not use Bazett's or other formulas when Rautaharju is requested.

    INSTRUCTIONS:
    - Identify the exact variables required from the text.
    - Apply the correct formula step-by-step.
    - Output your final evaluated numerical result enclosed in <answer> tags, like <answer>42.5</answer>.
    """




def _extract_number(value) -> Optional[float]:
    if value is None:
        return None
    if isinstance(value, (int, float)):
        return float(value)
    s = str(value)
    if s.startswith('**exception'):
        return None
    try:
        return float(s)
    except ValueError:
        pass
    for pat in (r'<answer>\s*([\d.eE+-]+)\s*</answer>', r'ANSWER:\s*([\d.eE+-]+)'):
        m = re.search(pat, s)
        if m:
            try:
                return float(m.group(1))
            except ValueError:
                pass
    nums = re.findall(r'-?\d+\.?\d*', s)
    if nums:
        try:
            return float(nums[-1])
        except ValueError:
            pass
    return None
