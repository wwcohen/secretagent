An analysis of the failures reveals two major issues dragging down the pipeline's accuracy:

1. **Hallucination in Intersection Computation:** The `find_available_slots` tool was using an LLM to compute the intersection of schedules. This is a deterministic math/logic problem, and the LLM frequently hallucinates overlaps or misses valid 30-minute gaps (e.g., in Case 418 it selected `11:00-11:30` even though Peter had a meeting starting at `10:30`).
2. **Missing Negative Constraints:** The `parse_schedules` LLM prompt did not clearly explain how to handle negative constraints like `"would rather not meet on Monday"` or `"avoid more meetings... after 14:30"`. As a result, these constraints were often ignored entirely or misclassified as a "preference".

### Strategy
1. **Convert `find_available_slots` and `select_and_format` to Pure Python:** We can add `method="direct"` to their `@interface` decorators. This forces the agent's tool call to run our exact Python logic instead of spinning up another error-prone LLM call. The Python logic deterministically checks 30-minute intervals against all extracted busy blocks.
2. **Improve the `parse_schedules` Docstring:** Add explicit instructions instructing the LLM to translate negative constraints directly into full `[start, end]` busy blocks on the given days.
3. **Fix `calendar_workflow` Preference Bug:** The original code hardcoded `"earliest"` when calling `select_and_format`. We update it to read the `preference` key from the extracted JSON with a fallback.

"""Task-specific interfaces for NaturalPlan calendar scheduling.

Decomposition derived from LLM reasoning traces:
1. parse_schedules — extract participants, busy slots, duration, preference
2. find_available_slots — compute free slot intersections
3. select_and_format — pick best slot, format answer
"""

from secretagent.core import interface
import json


@interface
def parse_schedules(problem_text: str) -> str:
    """Parse a calendar scheduling problem into structured JSON.

    Args:
        problem_text: The ENTIRE scheduling problem description as a single string.

    Returns:
        A JSON string encoding a dict with these keys:
        - participants: list of participant names
        - duration_minutes: meeting duration (30 or 60)
        - working_hours: [start_str, end_str] e.g. ["9:00", "17:00"]
        - days: list of day names e.g. ["Monday", "Tuesday"]
        - preference: "earliest" or "latest" or null
        - schedules: dict mapping participant name to dict mapping day to
          list of busy intervals e.g. {"Alice": {"Monday": [["9:00","10:00"]]}}

        CRITICAL INSTRUCTIONS FOR BUSY SCHEDULES:
        1. Explicit meetings: Add all explicit meeting times to the participant's busy schedule.
        2. Negative constraints: If a participant "would rather not meet", "do not want to meet", "can not meet", or "would like to avoid more meetings" on a specific day or time, YOU MUST TREAT THAT ENTIRE TIME PERIOD AS BUSY and add it to their schedule.
           - "Alice do not want to meet on Monday" -> add ["9:00", "17:00"] to Alice's Monday schedule.
           - "Bob would rather not meet on Tuesday after 13:30" -> add ["13:30", "17:00"] to Bob's Tuesday schedule.
           - "Charlie can not meet on Wednesday before 12:00" -> add ["9:00", "12:00"] to Charlie's Wednesday schedule.
           - "David would like to avoid more meetings on Thursday. Friday." -> add ["9:00", "17:00"] to David's Thursday AND Friday schedules.
        3. Preferences: If the group wants to meet at their "earliest availability", set preference to "earliest". If they prefer "latest", set preference to "latest". Otherwise null.

        Use 24-hour format. For participants with wide-open calendars, use empty lists.
    """


@interface(method="direct")
def find_available_slots(problem_text: str, schedules_json: str) -> str:
    """Find valid meeting slots that work for all participants."""
    def time_to_mins(t_str):
        try:
            h, m = map(int, str(t_str).strip().split(':'))
            return h * 60 + m
        except:
            return 0
            
    def mins_to_time(m_int):
        return f"{m_int // 60}:{m_int % 60:02d}"
        
    if isinstance(schedules_json, dict):
        data = schedules_json
    else:
        try:
            s = str(schedules_json).strip()
            if s.startswith("```json"):
                s = s[7:-3].strip()
            elif s.startswith("```"):
                s = s[3:-3].strip()
            if s.startswith("'") and s.endswith("'"):
                s = s[1:-1]
            data = json.loads(s)
        except Exception:
            return "[]"
            
    duration = data.get("duration_minutes", 30)
    wh = data.get("working_hours", ["9:00", "17:00"])
    if not wh or len(wh) != 2:
        wh = ["9:00", "17:00"]
    work_start = time_to_mins(wh[0])
    work_end = time_to_mins(wh[1])
    days = data.get("days", [])
    schedules = data.get("schedules", {})
    
    available_slots = []
    
    for day in days:
        for start_m in range(work_start, work_end - duration + 1, 30):
            end_m = start_m + duration
            conflict = False
            for person, person_schedule in schedules.items():
                busy_intervals = person_schedule.get(day, [])
                for interval in busy_intervals:
                    if not isinstance(interval, list) or len(interval) != 2:
                        continue
                    b_start = time_to_mins(interval[0])
                    b_end = time_to_mins(interval[1])
                    if b_start < end_m and b_end > start_m:
                        conflict = True
                        break
                if conflict:
                    break
            if not conflict:
                available_slots.append({
                    "day": day,
                    "start": mins_to_time(start_m),
                    "end": mins_to_time(end_m)
                })
                
    return json.dumps(available_slots)


@interface(method="direct")
def select_and_format(slots_json: str, preference: str = "earliest") -> str:
    """Pick the best slot based on preference and format the final answer."""
    if isinstance(slots_json, list):
        slots = slots_json
    elif isinstance(slots_json, dict):
        slots = [slots_json]
    else:
        try:
            s = str(slots_json).strip()
            if s.startswith("```json"):
                s = s[7:-3].strip()
            elif s.startswith("```"):
                s = s[3:-3].strip()
            if s.startswith("'") and s.endswith("'"):
                s = s[1:-1]
            slots = json.loads(s)
        except Exception:
            return "No available slots found."
            
    if not slots:
        return "No available slots found."
        
    slot = slots[0]
    if preference and isinstance(preference, str) and "latest" in preference.lower():
        slot = slots[-1]
        
    return f"Here is the proposed time: {slot['day']}, {slot['start']} - {slot['end']}"

def calendar_workflow(prompt: str) -> str:
    schedules_str = parse_schedules(prompt)
    
    preference = "earliest"
    try:
        data = json.loads(schedules_str)
        if data.get("preference"):
            preference = data.get("preference")
